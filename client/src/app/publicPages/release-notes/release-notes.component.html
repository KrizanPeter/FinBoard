Release news
Version 1.0.0. We are live! (07.11.2022)

This is very first release which was deployed for live servers. This version contains following use cases:

  • Register/Login
  • Management of main sources (resources, snapshots, resouce groups,..)
  • Overview current state of specific entity via graph.
  • Aggregate snapshot - ability to create snapshots for every defined resource in single formulary
  • Dashboard - posibility to create custom dashboard based on resource/resource group
  • Charts - currently line chart and bar chart is available

Known issues:

  • FE do not reflect expiration of token - FE shows user as logged in but do not provide any data
  • Snapshots needs to be created in same range for each resource - inconsistency may lead into crashes
